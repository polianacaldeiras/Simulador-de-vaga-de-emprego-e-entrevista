# Simulador-de-vaga-de-emprego-e-entrevista
Prompt para entrevista de emprego e diagnostico de pontos de melhoria para vagas de interesse com auxílio de IA

Prompt fornecido pelo professor Henrique Souza da DIO para definição da vaga e simulação de entrevista:

Você é um entrevistador técnico especializado em vagas de tecnologia.
SEU OBJETIVO:
Conduzir uma entrevista estruturada sobre uma vaga, fazendo perguntas uma por vez sobre 4 temas.
Após cobrir todos os temas e receber confirmação do usuário, gerar um resumo analítico.
IMPORTANTE: Faça apenas 1 pergunta por vez. Aguarde a resposta antes de prosseguir.
SEQUÊNCIA DE PERGUNTAS (nesta ordem):
- TÍTULO: "Qual é o título da vaga e qual o propósito principal desse cargo?"
- SENIORIDADE: "Qual a senioridade esperada e por quê?"
- STACK: "Quais tecnologias, frameworks e práticas são essenciais?"
- SOFT SKILLS: "Quais comportamentos ou atitudes são mais valorizados?"
REGRAS:
- Nunca faça mais de 1 pergunta por vez
- Só gere o resumo após confirmação explícita
INICIE COM:
"Olá! Vou fazer perguntas sobre a vaga que você está estruturando. Para começar: qual é o título da vaga e qual o propósito principal desse cargo?"


Resumo da vaga:

Resumo da Vaga – Dev Jr C#
Título e Propósito:
Desenvolvedor Júnior em C#, responsável por apoiar na construção e manutenção de aplicações, contribuindo para o desenvolvimento de soluções e adquirindo experiência prática na área.
Senioridade:
Nível Júnior – voltado para profissionais em início de carreira, com oportunidade de aprendizado e crescimento técnico sob orientação de colegas mais experientes.
Stack Tecnológico:
- Linguagem: C#
- Framework: Entity Framework
- Banco de dados: SQL
Soft Skills Valorizados:
- Comunicação clara para facilitar o trabalho em equipe
- Transparência no compartilhamento de informações e alinhamento de expectativas



Aqui está um diagnóstico claro sobre os pontos que você precisa evoluir para aumentar suas chances de conquistar uma vaga como Dev Jr C#:

Pontos Fortes
- Base em programação: experiência com C e Visual C++ dá uma boa fundação lógica.
- Atitude de aprendizado: dedicação aos estudos e busca de apoio com colegas mais experientes.
- Soft skills sólidas: comunicação e transparência já demonstradas em experiências profissionais anteriores.
Pontos a Evoluir
- Domínio do C# e .NET
- Praticar conceitos fundamentais: orientação a objetos, classes, interfaces, herança, polimorfismo.
- Explorar o ecossistema .NET (ASP.NET Core, Windows Forms ou WPF, dependendo do foco).
- Banco de Dados e SQL
- Aprender comandos básicos de SQL (SELECT, INSERT, UPDATE, DELETE).
- Entender modelagem de dados e relacionamentos.
- Entity Framework
- Estudar como funciona o mapeamento objeto-relacional (ORM).
- Praticar consultas LINQ e operações CRUD com EF.
- Projetos Práticos
- Criar pequenos sistemas (ex.: cadastro de usuários, controle de tarefas) para aplicar C#, SQL e EF juntos.
- Publicar no GitHub para mostrar evolução e prática real.
- Cultura de Desenvolvimento
- Familiarizar-se com boas práticas: versionamento com Git, testes básicos, princípios SOLID.
- Participar de comunidades ou fóruns para trocar experiências.

Diagnóstico Final
Você já tem boa base lógica e soft skills alinhadas ao perfil da vaga. O que falta é construir experiência prática em C#, SQL e Entity Framework, mostrando que consegue aplicar esses conhecimentos em projetos reais. Com dedicação a estudos direcionados e prática constante, você terá condições sólidas para conquistar uma vaga de Dev Jr C# em pouco tempo.
